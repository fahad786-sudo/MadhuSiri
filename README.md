# 🐝 MadhuSiri – Bee Farmer Harmony App

MadhuSiri is an Android application developed using Kotlin, Firebase, Google Maps, and Generative AI to help protect bees from harmful pesticide spraying in nearby agricultural fields.

Built as part of the **MindMatrix VTU Internship Program** under the project:

## Android App Development using GenAI – Madhu-Siri (Agriculture)

---

# 📌 Problem Statement

Beekeeping (Apiculture) is an important source of income, but thousands of bees die due to pesticide spraying in nearby farms.

Beekeepers often do not know:
- when farmers spray pesticides
- where spraying happens
- how to protect active bee colonies

MadhuSiri solves this problem by creating communication between:
- 🧑‍🌾 Farmers
- 🐝 Beekeepers

Farmers can send spray alerts, and beekeepers receive notifications to temporarily protect their hives.
